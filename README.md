# EchoSMs 2026 workshop material

This repository contains presentations, notebooks, and code for the echoSMs
anatomical datastore workshop, held online and in person on 19 April 2026, prior to the 2026 ICES WGFAST meeting.

The main presentation for the workshop is part of the echoSMs documentation, available [here](https://ices-tools-dev.github.io/echoSMs/training/2026%20WGFAST%20workshop/overview/).
